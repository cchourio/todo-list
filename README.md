# TODO List

A simple task management application built with React and Vite. TODO List lets you keep track of the things you need to get done — add tasks, mark them complete, and stay organized.

## Author

Christian Chourio

## Tech Stack

- [React](https://react.dev/) — UI library
- [Vite](https://vitejs.dev/) — build tool and development server

## Installation

Clone the repository, then run the following commands inside the project folder:

```bash
# Scaffold the Vite + React app inside your cloned (empty) repo:
npx create-vite@latest --template react .

# Install dependencies:
npm install
```

## Running the Development Server

Start the local development server with:

```bash
npm run dev
```

Once it's running, open the URL shown in your terminal (usually `http://localhost:5173`) in your browser to view the app.
